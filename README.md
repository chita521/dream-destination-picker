# Dream Destination Picker

## Overview

Dream Destination Picker is a Python GUI application built with Flet that helps users discover travel destinations based on their preferences.

Users select a budget, weather preference, and activity type. The application then recommends a destination, suggests an activity, provides a packing reminder, and displays an image of the destination. Users can also save their trip recommendation and view their most recently saved destination.

## Features

* Select a travel budget
* Choose a preferred weather type
* Choose an activity type
* Receive a destination recommendation
* View a destination image
* Get a suggested activity
* Receive a packing reminder
* Save trip recommendations
* Load previously saved trip information
* Multi-page interface with navigation

## Technologies Used

* Python
* Flet
* JSON
* GitHub

## Project Structure

* `main.py` – Main GUI and page navigation
* `controller.py` – Application logic and file handling
* `data.py` – Destination recommendation data
* `pages/` – Individual page layouts
* `resources/` – Images and fonts
* `saved_trip.json` – Persistent saved trip data

## How to Run

Open the project in VS Code and run `main.py`
